# 遠端存取偏好：SSH 優先

## 規則

需要讓使用者遠端存取服務或介面時：

- **優先**：開放 SSH port，讓使用者透過 SSH tunnel 或直連存取
- **避免**：預設啟動 VNC、RDP 或其他 GUI 桌面串流服務

## 禁止行為

- 不得在未說明理由的情況下，預設提供 VNC/RDP 解法

## 適用情境

| 情境 | 建議方案 |
|------|----------|
| 遠端執行指令、傳輸檔案、查看日誌 | SSH |
| 服務本身無法在非圖形環境運作（桌面應用、瀏覽器視覺除錯）| 可選 VNC，需先說明原因並確認 |

## 例外

若服務本身需要圖形介面，可提供 VNC，但必須：
1. 說明為何 SSH 無法滿足需求
2. 取得使用者確認後再啟動
