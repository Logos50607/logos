# 清單格式規範 (List Format Guidelines)

當編輯 `AGENT_PLAN.md` (或 `TODOLIST.md`) 與 `ASK_HUMAN.md` (或 `ASKHUMAN.md`)
時，必須遵循以下格式：

- **格式規範**：
  - 使用條列表 (`- [ ]`, `- [x]`)。
  - 標號後加上日期 (格式：`YYYY-MM-DD`)。
  - **最新日期優先**：新加入的項目應置於文件最上方。
