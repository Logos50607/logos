# Project Dependencies

本專案所有的開發工具與外部套件依賴記錄如下：

## 1. 核心系統工具

| 依賴名稱 | 用途                           | 安裝/檢查方式 |
| :------- | :----------------------------- | :------------ |
| `curl`   | 網路請求、擷取網頁內容         | `which curl`  |
| `sed`    | 流編輯器，用於文本轉換         | `which sed`   |
| `awk`    | 文本處理語言，用於清理與格式化 | `which awk`   |
